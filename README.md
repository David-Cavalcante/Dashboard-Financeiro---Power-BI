## 📌 Sobre o Projeto

Este projeto apresenta um Dashboard Financeiro desenvolvido em Power BI, criado para acompanhar indicadores financeiros estratégicos de uma organização.

O painel permite monitorar receitas, custos, impostos, lucro e margem financeira, além de visualizar a evolução dos resultados ao longo do tempo.

O objetivo é fornecer uma visão clara da saúde financeira do negócio, facilitando análises gerenciais e apoiando decisões baseadas em dados.

🎯 Principais Indicadores (KPIs)

O dashboard apresenta indicadores financeiros fundamentais:

💵 Recebimentos

Valor total recebido pela empresa no período analisado.

Exemplo no dashboard:
R$ 94.60M

💸 Custos

Total de despesas operacionais relacionadas às atividades da empresa.

Exemplo:
R$ 44.86M

🧾 Impostos

Valor total pago em tributos.

Exemplo:
R$ 14.19M

📈 Lucro

Resultado final após dedução de custos e impostos.

Exemplo:
R$ 35.55M

## 🏦 Instituições Financeiras

O dashboard também apresenta os bancos utilizados nas operações financeiras:

Nubank

Essa visualização pode ser utilizada para acompanhar movimentações financeiras por instituição.

## 💳 Análise de Movimentações

O painel apresenta uma análise das transações realizadas.

Exemplo exibido:

2725 movimentações financeiras

1200 realizadas via Pix

44.04% das transações via Pix

Esse indicador permite analisar a adoção de meios de pagamento digitais.

📅 Evolução do Lucro por Mês

O dashboard apresenta um gráfico de Waterfall (cascata) mostrando o impacto das variações financeiras mês a mês.

Esse gráfico permite identificar:

meses com maior crescimento financeiro

períodos de queda no resultado

impacto acumulado ao longo do ano

com analises Mensais.

## 🧠 Insights que o Dashboard Permite

Com este painel é possível identificar:

evolução do resultado financeiro

impacto de custos e impostos no lucro

margem financeira do negócio

meses com melhor performance

adoção de meios de pagamento digitais

distribuição das operações bancárias
